set_border(screen, ls=0, rs=0, ts=0, bs=0, tl=0, tr=0, bl=0, br=0)¶

This is the border function of curses.border Draw a border aorund the edges of the window. Each parameter specifies the character to use for a specific part of the border.

Parameters:
  • screen (curses.screen) – curses.screen
  • ls (int) – Left Side
  • rs (int) – Right side
  • ts (int) – Top
  • bs (int) – Bottom
  • tl (int) – Upper-left corner
  • tr (int) – Upper-right corner
  • bl (int) – Bottom-left corner
  • br (int) – Bottom-right corner
